## Objective
Demonstrate Joiner, Mover, and Leaver (JML) access management using Microsoft Entra ID and Jira ticketing workflows using a real world job description for an IAM Governance Analyst (Sceenshot provided in the folder). 

## Tools Used
- Microsoft Entra ID (Free Tier)
- Jira (Project Management)
- GitHub (Documentation)

## Scenarios Covered
- Joiner: New Hire Access Provisioning
- Mover: Role Change Access Modification
- Leaver: User Deprovisioning & Access Removal

## Workflow
1. HR submits access request (Joiner)
2. IAM reviews and provisions role-based access
3. Access modified based on job role changes (Mover)
4. Access revoked and account disabled (Leaver)

## Key IAM Concepts
- RBAC
- Least Privilege
- Identity Lifecycle Management
- Access Governance
